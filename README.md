# Jarvis-Voice-Assistant
Jarvis Voice Assistant

Jarvis is a Python-based voice assistant that can listen to your voice commands and perform various tasks such as opening websites, telling jokes, searching Wikipedia, playing YouTube videos, and more.

Features
🎙️ Speech recognition (understands voice commands)
🗣️ Text-to-speech response
🌐 Search on Wikipedia
📺 Play YouTube videos
😂 Tell jokes
📅 Give date and time
🔍 Google search support
📌 Easy to extend with more features


Installation
1. Clone the repository
git clone https://github.com/anushkagaur22/Jarvis-Voice-Assistant-clean.git
cd Jarvis-Voice-Assistant

2. Create virtual environment (optional but recommended)
python -m venv venv
venv\Scripts\activate   # On Windows
source venv/bin/activate   # On Linux/Mac

3. Install dependencies
pip install -r requirements.txt

Usage

Run the main script:
python jarvis.py

Then say "Jarvis" followed by your command, for example:
“Jarvis, play music on YouTube”
“Jarvis, tell me a joke”
“Jarvis, search Python on Wikipedia”
